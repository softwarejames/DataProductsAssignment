Automobile MPG Estimate

Data Products Course Project

Overview

  • The data was extracted from the 1974 Motor Trend US magazine comprised of fuel consumption and 10 aspects of automobile design and performance for 32 automobiles (1973â€“74 models).
  • A linear regression model was built to estimate MPG using transmission type, number of cylinders, and horsepower variables in this data set.
  • A Shiny App was created that allows users to input transmission type, number of cylinders, and horsepower and receive an estimate of MPG based on this model.
